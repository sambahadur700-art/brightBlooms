
# Bright Bloom 🌸

Bright Bloom is a modern, responsive frontend website designed for an early childhood learning center. The website focuses on providing a clean, engaging, and user-friendly experience for parents while showcasing educational programs and activities.

## ✨ Features

- Responsive Design
- Modern UI/UX
- Smooth Scroll Animations
- Interactive Navigation
- Image Gallery
- Booking Modal
- Mobile-Friendly Layout
- Clean and Organized Code

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)
